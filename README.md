# GomiDashi Public Website

This repository contains static pages for App Store submission:

- `privacy.html`: Privacy Policy URL
- `support.html`: Support URL
- `index.html`: simple landing page linking to both

## Recommended URLs

This repository is intended to be published from:

- Repository: `hasirqi/GomiDashi`
- GitHub Pages root: `https://hasirqi.github.io/GomiDashi/`

Use these URLs in App Store Connect:

- Privacy Policy URL: `https://hasirqi.github.io/GomiDashi/privacy.html`
- Support URL: `https://hasirqi.github.io/GomiDashi/support.html`
- Marketing URL: `https://hasirqi.github.io/GomiDashi/`
